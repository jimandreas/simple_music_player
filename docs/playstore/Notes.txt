  Before submitting, you'll need to:

  1. Change the package name from com.bammellab.musicplayer to your own unique identifier
  2. Host the privacy policy at a public URL (GitHub Pages works well)
  3. Create the feature graphic (1024x500) - see asset specs
  4. Capture 2+ more screenshots showing different features
  5. Fill in placeholder fields (email, website) in the documents
  6. Generate a signed release build: ./gradlew bundleRelease

  The submission checklist has everything you need to get accepted on the first try.
