# Release Notes

## Version 1.0.0 (Initial Release)

### What's New
First release of Music Player - a simple, clean folder-based music player.

### Features
- Browse and select folders using Android's file picker
- Play MP3, WAV, OGG, FLAC, M4A, and more
- Playback controls: play/pause, previous/next track
- Volume adjustment
- Shuffle mode
- Seek bar for track navigation
- Now Playing display with current track info
- Auto-scroll to current track

### Play Store "What's New" Text (500 characters max)

```
Welcome to Music Player!

A clean, simple music player for your local audio files.

Features:
- Select any folder to play music from
- Support for MP3, FLAC, WAV, OGG, M4A & more
- Play/pause, skip, volume controls
- Shuffle mode
- Seek bar navigation
- Modern Material 3 design

No ads. No tracking. Just music.
```

## Release Notes Best Practices

1. **Keep it concise** - Users skim release notes
2. **Lead with the most important change**
3. **Use bullet points** for readability
4. **Mention bug fixes** but don't be overly technical
5. **Stay within character limits** - 500 chars for Play Store
